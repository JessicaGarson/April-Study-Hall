# April Study Hall
This talk is a look at the projects, tools, and ideas I’ve been exploring recently, with a focus on live coding, music-making, and performance. I’m planning on showing you what I’ve been working on lately, including creative coding experiments, performance tools, and the process behind how I make songs. The talk will also introduce live coding as a practice, walk through some of the technologies used in that space, and include a demo of Strudel.

By the end of the talk, attendees will:

- Understand the basics of live coding
- Get exposure to tools like Strudel
- Learn more about my creative process
- See examples of how coding and music can intersect in performance
